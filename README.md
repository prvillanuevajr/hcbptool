
# HC BP TOOL
This tool aims to help N1 hc bp developers in managing files between repository and wcs workspace.

## Features
- Transfer file from Repository to WCS Workspace
- Transfer file from WCS Workspace to Repository
- Faster Set DB
- Manage Database Configurations

## Download
Go to the latest tag and download the executable file.
